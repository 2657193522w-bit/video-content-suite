## 🎣 开头Hook（0-3秒）

**核心目标**: 在前3秒抓住观众注意力

### 可选Hook类型:

#### 悬念型
- "看完这个，我再也不[痛点]了"
- "99%的人都不知道[话题]的真相"
- "原来[话题]才是[收益]的关键"

#### 数字型
- "[数字]个技巧，让你[收益]"
- "学会这[数字]招，[收益]很简单"
- "[数字]分钟学会[话题]"

#### 痛点型
- "为什么你总是[痛点]？"
- "如果你[痛点]，一定要看完"
- "[痛点]的解决方案，终于找到了"

#### 反常识型
- "专家不会告诉你的[话题]真相"
- "我反对所有人对[话题]的看法"
- "[话题]，你可能一直做错了"

## 📝 正文内容

**核心目标**: 提供价值，保持节奏

### 内容结构:

#### 抖音风格（快节奏）
```
第一，[要点1]。很多人不知道，其实[补充说明]。
第二，[要点2]。记住这一点，你就超过了90%的人。
第三，[要点3]。这个技巧特别实用。
```

#### B站风格（深度解析）
```
首先，我们需要了解[话题]的基本概念。
[要点1]。这里我要特别强调一下，很多人容易忽略这个细节。
[要点2]。从原理上来说，这是因为[解释]。
总结一下，[话题]的关键在于[核心观点]。
```

#### 小红书风格（生活化）
```
亲测分享！关于[话题]，我总结了几点经验：
✨ [要点1]
✨ [要点2]
✨ [要点3]
这些都是我自己实践过的，真的很有用！
```
